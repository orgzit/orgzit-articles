Orgzit provides its users with a collaborative work management platform and helps increase the efficiency of their business workflows. The Orgzit API is one of the ways to help your team achieve more together!

The Orgzit Web API is an interface for querying information _from_ and enacting change _in_ an Orgzit workspace. You can use the API to import external data into your account or trigger actions in Orgzit in response to a change in an external software or service.

You can use this integration to perform the following:

1.  [Create a Record](/articles/api-integration-how-to-create-records)
    
2.  [Update a Record](/articles/api-integration-how-to-update-records)
    
3.  [Fetch Records](/articles/api-integration-how-to-fetch-records)
