Within Orgzit, you can add data fields and assign attributes to them. An attribute decides how a data field will function. You can choose the right field attribute according to your requirement.  
  
[You can learn about data fields and attributes in this support article.](/articles/what-are-data-fields-and-field-attributes)  
  
​

Remember - Not all field attributes and data types can be used together. Check below to see which attributes can be applied to your data type.
