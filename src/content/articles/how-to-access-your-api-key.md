Your API Key is essential for authentication while setting up an API Integration.

To access your API Key, log into your Orgzit account, and go to your **User Profile** page.

![](/images/how-to-access-your-api-key/img-1.jpeg)

You will find your API Key in the **System Information** Section there.

Copy your entire API Key to use it for setting up an API Integration.

![](/images/how-to-access-your-api-key/img-2.jpg)
