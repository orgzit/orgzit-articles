# Orgzit User Primary Attributes  

Every user has the following primary attributes associated with their user data. The information includes: 

First Name  
Last Name  
User Photograph  
Email Address  
Phone Number  
Time Zone  
Password

The email address provided by the user is the same email address that the user wants to use to login the information. 

Every Orgzit user can be associated with one or multiple Orgzit Accounts. For example, Chris Bay can be associated with the following two accounts

Acme Inc. accessible for Acme Inc and Zed Solutions Inc respectively.

```
https://acme.orgzit.comhttps://zed-solutions.orgzit.com 
```

_Note: Irrespective of the account, the key primary user attributes remain the same._
