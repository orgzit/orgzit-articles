The show/hide fields feature enables you to choose which data fields or columns you want to see and which ones you don't. This functionality is also useful in case of creating [personalized views](/articles/how-to-create-custom-tabular-views).  
  
To show/hide fields in Orgzit following steps are to be followed:

1.  Just below the top left corner of the selected Workspace, there will be a button named "**Show/Hide Fields**".
    
2.  Clicking on that button will generate a drop-down menu which will have a list of all the fields present in the selected workspace. You can now select which fields to show or hide by checking/unchecking the field name checkbox.
    
3.  You can also select all or deselect all the fields in one go by clicking on the buttons named "Show all" or "Hide all" respectively.
